SPAD Pro — Analizador de Clorofila

Aplicación de escritorio para estimar el contenido de clorofila en hojas de mango 
mediante análisis de imagen en tiempo real, usando una cámara ESP32-CAM como fuente de video.
¿Cómo funciona?

El sistema captura 5 zonas distintas de una hoja y analiza cada recorte en el espacio
de color HSV para separar píxeles verdes (hoja sana) de amarillos (clorosis)
. Con esos porcentajes calcula un valor SPAD estimado por zona 
y luego promedia los 5 resultados para entregar un diagnóstico final.
